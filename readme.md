# Stencil-calculator

Calculadora creada con stencil

### Instalación 🔧
Para empezar a usar este componente:


* Clona este proyecto.
 
* Ve a la carpeta del proyecto:
```
cd calculator-stencil
```

* Instala las dependencias:
```
npm install
```
* Ejecuta el programa:
```
npm start
```


## Construido con 🛠️

* [Stencil](https://stenciljs.com)
